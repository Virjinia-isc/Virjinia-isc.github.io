Habilidades

Desarrollo de sw de escritorio

Desarrollé una aplicación de escritorio para la gestión de variables ambientales, utilizando C# para la programación y bases de datos SQL para el almacenamiento de información. El proyecto incluyó la integración de hardware mediante Arduino y sensores, permitiendo la captura y monitoreo en tiempo real de datos ambientales.

Desarrollo web

He sido instructora en Tecnolochicas brindado el curso "diseño de sitios web". Creé sitios web responsivos aplicando los conocimientos obtenidos durante mi carrera y el bootcamp de Tecnolochicas PRO. Algunos han sido: sitio web para gestion de anuncios publicitarios, sitio web para el monitoreo de jitomate, clon de pinterest y blogs.

IoT

Como académica, realicé prácticas enfocadas en la programación de dispositivos y sensores, utilizando las placas Arduino y Raspberry Pi 3. Empleando el desarrollo de sistemas para captura y monitoreo de datos en tiempo real, conexión de dispositivos con DB para almacenamiento y análisis (firebase) y diseño de aplicaciones para la automatización de procesos basados en datos recopilados por sensores.
